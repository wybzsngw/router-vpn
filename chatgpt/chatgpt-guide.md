## 一、ChatGPT 是什么

ChatGPT 是 OpenAI 开发的大语言模型（LLM）对话系统，基于 GPT-4o、GPT-4 等模型，能进行自然语言对话、写作、编程、翻译、数据分析等任务。

- **官网**：<https://chatgpt.com>
- **免费版**：可使用 GPT-4o mini，有频率限制
- **付费版（Go/Plus/Pro）**：解锁 GPT-4o、GPT-4、高级语音、图像生成等

## 九、常见问题 FAQ

**Q1：注册时提示 "OpenAI's services are not available in your country"？**

A：当前节点 IP 被识别为不支持的地区。切换到日本、新加坡或美国节点，清除浏览器 Cookie 后重试。

**Q2：登录后响应慢或经常断开？**

A：换低延迟节点（推荐日本/新加坡），或升级机场套餐获取更快速度。

**Q3：回答到一半停了？**

A：输入「继续」或「continue」，ChatGPT 会接着上次的位置继续。

**Q4：收不到验证码邮件？**

A：检查垃圾箱；换用 Gmail 或 Outlook 邮箱。

**Q5：Plus 支付失败怎么办？**

A：确保 IP 与账单地址一致，使用美国节点 + 虚拟信用卡，或改用 Apple 订阅方式。详见 [Plus 充值指南](./chatgpt-plus-guide.md)。

**Q6：中国区搜不到 ChatGPT App？**

A：需要外区 Apple ID，参考 [手机教程](../mobile/ios-shadowrocket.md) 中的外区 ID 注册说明。

**Q7：免费版够用吗？**

A：轻度使用（每天几次问答）够用。如果需要频繁使用、编程辅助或图像生成，建议升级 Go 或 Plus。

**Q8：能买成品号吗？**

A：可以。[账号星球](https://aixiaobai.accboy.com) 提供 ChatGPT 成品账号（¥48.8 起）和 Plus 账号（¥98 起），支持支付宝。但成品号有被找回、封号等风险，长期使用仍建议自行注册。

**Q9：Plus 与 API 是否一回事？**

A：不是。Plus 是网页端会员，API 是按量付费的开发者接口，两套计费独立。详见 [OpenAI API 指南](./openai-api-guide.md)。
