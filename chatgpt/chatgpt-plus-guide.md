## 一、Go、Plus、Pro 怎么选

| 方案 | 月费（参考） | 核心功能 | 适合人群 |
|------|------|----------|----------|
| **Free** | $0 | GPT-5.5 Instant 有限额度、有限上传/图像/深度研究 | 偶尔体验 |
| **Go** | 约 $8 | 更多 GPT-5.5 Instant、更多消息和上传、更长记忆 | 高频轻量用户 |
| **Plus** | 约 $20 | GPT-5.5 Thinking、Codex 更高额度、Deep Research、项目/任务/自定义 GPT | 工作、学习、编程主力用户 |
| **Pro** | 约 $100 | GPT-5.5 Pro、5–20 倍配额、GPT-5.3 及文件上传不设限 | 重度用户、专业开发者 |

> 💡 价格因地区而异。OpenAI 新加坡区页面显示 Go S$11/月、Plus S$30/月、Pro S$138/月；美元价格按常见近似值写作，最终以结账页为准。大多数普通用户选 **Go** 即可满足日常轻量需求；需要推理模型、Codex 或 Deep Research 选 **Plus**。

## 二、支付前必须满足的 3 个条件

1. **能稳定打开** <https://chatgpt.com> 并完成登录
2. **OpenAI 账号**：已完成注册并能正常登录
3. **支付工具**：国际信用卡（Visa/Mastercard）或虚拟信用卡，或 iPhone + 外区 Apple ID

## 四、网页版升级 Plus 的具体步骤

1. 登录 <https://chatgpt.com>
2. 点击左下角头像 → 「My Plan」→「Upgrade」
3. 选择 Go 或 Plus 套餐
4. 填写信用卡信息（卡号、有效期、CVV、账单地址）
5. 账单地址尽量与卡片发行地区保持一致
6. 点击确认支付

> ⚠️ **注意**：不要在短时间内连续重复提交付款，可能触发风控。

## 五、iPhone 通过 Apple 订阅

1. 在 iPhone 上登录支持 ChatGPT 的 **外区 Apple ID**（常见为美区/新加坡区等）
2. 根据对应地区支持的方式充值或绑定付款方式（礼品卡、当地银行卡、PayPal 等以 App Store 实际页面为准）
3. 下载 ChatGPT App（外区 App Store）
4. 打开 App → 登录 → Settings → Subscription → 选择套餐
5. 通过 Apple 内购完成支付

## 六、支付失败最常见的原因

1. **网络不稳定**：支付过程中网络中断或 IP 变动，可能触发风控 → 保持网络稳定后重试
2. **卡片被拒**：虚拟卡余额不足或被风控 → 换一张卡
3. **浏览器缓存**：旧 Cookie 干扰 → 用无痕模式重试
4. **账单地址不匹配**：与卡片地区不一致 → 尽量统一地区

> **反复失败怎么办？** 不建议把主账号交给第三方代操作。可以先换浏览器无痕模式、改用 Apple 订阅，或者用 Cursor Pro 调用 GPT-5.5 / Codex 等模型完成文档和代码任务。

## 七、常见问题 FAQ

**Q1：Go 值不值得买？**

A：值得。如果你每天都用 ChatGPT 做问答、翻译、文案，Go（约 $8/月）比免费版额度更高，性价比很好。但如果你需要 GPT-5.5 Thinking、Codex 或 Deep Research，建议直接选 Plus。

**Q2：Plus 与 API 是否一回事？**

A：不是。Plus 是网页端会员，API 是按量付费的开发者接口，两套独立计费。详见 [OpenAI API 指南](./openai-api-guide.md)。

**Q3：网页端失败能否改 Apple 订阅？**

A：可以。不少用户反馈 Apple 订阅更顺利，但需要外区 Apple ID 和礼品卡。

**Q4：能不能把账号交给第三方处理付款？**

A：不建议作为长期方案。把账号交给第三方可能涉及账号找回、共享污染、隐私泄露和付款争议。长期使用 AI，优先自己注册并绑定自己的付款方式；如果只是想用 GPT-5.5 / Codex 处理文档和代码，也可以考虑通过 Cursor IDE 使用。
