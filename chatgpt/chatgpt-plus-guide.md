## 四、网页版升级 Plus 的具体步骤

1. 登录 <https://chatgpt.com>，确保科学上网节点为美国
2. 点击左下角头像 → 「My Plan」→「Upgrade」
3. 选择 Go 或 Plus 套餐
4. 填写信用卡信息（卡号、有效期、CVV、账单地址）
5. 账单地址建议填写美国免税州地址（如 Oregon）
6. 点击确认支付

> ⚠️ **注意**：不要在短时间内连续重复提交付款，可能触发风控。

## 五、iPhone 通过 Apple 订阅

1. 在 iPhone 上登录 **外区 Apple ID**（推荐美区）
2. 充值 Apple 礼品卡（可在淘宝、亚马逊购买）
3. 下载 ChatGPT App（外区 App Store）
4. 打开 App → 登录 → Settings → Subscription → 选择套餐
5. 通过 Apple 内购完成支付

## 六、支付失败最常见的原因

1. **节点不稳定**：支付过程中 IP 变动，触发风控 → 全程使用同一节点
2. **卡片被拒**：虚拟卡余额不足或被风控 → 换一张卡
3. **浏览器缓存**：旧 Cookie 干扰 → 用无痕模式重试
4. **账单地址不匹配**：与 IP 地区不一致 → 统一使用美国地址和美国节点
