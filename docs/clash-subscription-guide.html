Checklist

通用申请要点

建议
  • 优先选择支持 Clash 订阅 与 HTTPS 的机场。
  • 留意是否提供「订阅转换/短链」与「流媒体解锁/ChatGPT」。
  • 保存售后渠道（客服/工单/Telegram 群）。
  • 订阅链接一般可在「我的订阅 / 账户 / 仪表盘」中找到。
  • 导入后如节点为空，先在机场后台「重置订阅」再刷新。
Import

将订阅导入 MerlinClash（华硕梅林）

  1. 进入路由器管理界面 → 软件中心 → MerlinClash → 打开「订阅设置」。
  2. 在「订阅链接」粘贴机场提供的 Clash 订阅 URL（https://...）。
  3. 点击「更新订阅」→ 等待节点同步完成。
  4. 到「运行状态/节点列表」检查是否加载成功；如失败，重试「更新订阅」。
提示：若需分流/解锁流媒体，可在「规则设置」启用对应的规则集或策略组。
