📖 进阶阅读：代理能做什么、不能做什么：AI 访问与住宅 IP 真相 —— 读完可以了解：代理只负责跨国传输、机房 IP 与住宅 IP 的区别，以及为什么换 IP 不一定能解决 AI 项目（如 Claude Code）的访问 / 封号问题。
