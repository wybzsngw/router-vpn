方案 B · 进阶

方案 B：FlClash / ClashMeta + 回国节点分流

如果你已经有 Clash 内核客户端和回国节点来源（机场回国专线 / 大陆家宽出口），可以在现有配置里直接追加回国分流规则。

⚠️ 前提

商业加速器（Malus / 快帆）多数是封闭 App，不提供可导入 Clash 的节点链接，不能走此方案。需要从支持导出节点的机场或自建大陆家宽出口获取回国节点。

3.2 在配置文件中追加回国规则

打开你现有的 Clash 配置文件（.yaml），分两步追加：

步骤 1：添加 rule-providers

rule-providers:
  return-china:
    type: http
    behavior: classical
    format: yaml
    url: "https://raw.githubusercontent.com/mowei-ie/router-vpn/main/docs/return-china-rules.yaml"
    path: ./ruleset/return-china.yaml
    interval: 86400

步骤 2：在 rules 最前面添加（优先级最高）

rules:
  - RULE-SET,return-china,🇨🇳回国          # 放在所有规则最前面
  - GEOSITE,cn,🇨🇳回国                      # 可选：通用大陆兜底
  - GEOIP,CN,🇨🇳回国,no-resolve             # 可选：大陆 IP 兜底
  # --- 你原有的出国规则继续在后面 ---
  - GEOSITE,google,出国代理
  - MATCH,出国代理

步骤 3：新增「🇨🇳回国」代理组

proxy-groups:
  - name: "🇨🇳回国"
    type: select
    proxies:
      - 回国节点A
      - 回国节点B
      - DIRECT

3.3 在 FlClash 中导入配置

  1. 底部导航 → Profile（配置） → + → 选择本地文件或 URL 导入

  2. 点击导入的配置条目 → 确认激活（变为蓝色高亮）

  3. 主页开关打开 VPN → 在「🇨🇳回国」代理组选择具体的回国节点（不能选 DIRECT，否则走本地出网）

  4. 代理模式确认为 Rule（规则），不要用 Global

3.4 在 CMFA 中导入配置

  1. 主页 → 配置 → + → 从 URL 导入 / 从本地文件导入

  2. 长按配置条目 → 选为当前配置

  3. 主页 Enable TUN 开关打开 → 代理模式选 Rule（规则）

  4. 在代理页 → 找到「🇨🇳回国」代理组 → 选择具体的回国节点

方案 C · 有大陆落地点

方案 C：Tailscale 客户端（大陆家宽出口）

如果你已经按照 大陆家宽 Tailscale Exit Node 教程 配好了大陆出口设备：

  1. Google Play / APK 安装 Tailscale Android App

  2. 登录与大陆出口设备同一个 Tailscale 账户

  3. App 主界面 → 点 Exit Node → 选择大陆那台设备

  4. 手机所有流量从大陆家宽真实 IP 出网 → 锁区内容自然解锁

⚠️ 注意

Tailscale 全局出口会让所有流量（包括访问 YouTube / Google 等海外服务）都绕回大陆，会显著增加海外网站访问延迟。如需同时流畅访问海外服务，建议用方案 B 的 FlClash 分流（精确控制哪些流量走回国），而不是 Tailscale 全局出口。

FAQ

常见问题

Q1：Malus / 快帆 和 FlClash 可以同时开吗？

不可以。Android 同一时间只允许一个 VPN 连接，开第二个会自动断开第一个。选其中一种方案使用即可。

Q2：FlClash 和 ClashMeta for Android 哪个更好？

推荐 FlClash：UI 更现代，Material 3 设计，规则集自动刷新逻辑更完善，新项目维护活跃。如果你原来就用 CMFA，继续用也没问题。

Q3：机场没有回国专线，哪里获取回国节点？

两条路：① 联系机场确认是否提供 IEPL/IPLC 双向回国线路；② 在大陆有常开设备的话，用 Tailscale Exit Node 自建（参考方案 C）。若两条路都不具备，直接用方案 A 商业加速器。

Q4：海外 VPS（CN2 GIA）能作为回国节点解锁爱奇艺吗？

不能。海外 VPS 出口 IP 在海外，爱奇艺 / 央视频 / B 站等锁区内容只认大陆 IP。把海外 VPS 加进「🇨🇳回国」代理组，锁区内容仍会报"该内容仅限中国大陆"，不会有任何改善。

Q5：配置回国分流后，访问淘宝 / 微信速度变慢了？

正常现象。流量绕经了回国节点（可能有额外延迟），取决于节点线路质量。若节点质量差，访问大陆服务反而会比直连慢。解决方法：换更好的回国节点，或对影响不大的服务在规则里改为 DIRECT。
