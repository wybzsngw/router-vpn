Section 01

ChatGPT 是什么

ChatGPT 是 OpenAI 开发的大型语言模型对话系统，能够理解自然语言并生成高质量文本。自 2022 年 11 月发布以来，已成为全球最受欢迎的 AI 工具之一，广泛应用于：

  • 写作辅助：撰写文章、邮件、报告、文案、翻译
  • 编程开发：代码编写、调试、代码审查、技术文档
  • 学习研究：概念解释、论文辅助、外语学习、知识问答
  • 创意设计：AI 绘图（DALL-E）、头脑风暴、内容策划
  • 数据分析：上传文件进行数据处理、图表生成、报告撰写
  • 日常助手：旅行规划、食谱推荐、职业建议等

2026 年 5 月，OpenAI 主推 GPT-5.2（官方定位：为专业工作、编程和长时运行智能体而打造，已向所有人开放）。模型按能力分为三档：GPT-5.5 Instant（日常快速，免费/Go 可用）→ GPT-5.5 Thinking（高级推理，Plus 起）→ GPT-5.5 Pro（专业推理，Pro 专属）。付费用户还可使用 Codex 智能体和 Deep Research 深度研究模式。除了在 ChatGPT 网页端使用，你也可以通过 Cursor IDE 直接调用 OpenAI 最新模型——这也是目前性价比最高的使用方式之一，详见第八节。

Section 03

注册 OpenAI 账号

OpenAI 在 2024 年底已大幅简化注册流程，不再强制要求海外手机号验证。以下是当前最新注册方法：

方法一：邮箱注册（推荐）

  1. 确保已连接到日本/新加坡/美国等节点
  2. 访问 chatgpt.com
  3. 点击 Sign up（注册）
  4. 输入邮箱地址（推荐使用 Gmail / Outlook / ProtonMail）
  5. 设置密码（至少 12 位，建议包含大小写字母和数字）
  6. 到邮箱中查收验证邮件，点击验证链接
  7. 填写姓名和生日信息，完成注册

方法二：第三方账号快捷登录

在注册页面可直接选择以下账号一键登录：

  • Google 账号 — 最方便，推荐
  • Microsoft 账号 — 适合已有 Outlook 邮箱的用户
  • Apple ID — 支持隐藏邮箱功能
注册建议：优先使用 Gmail 注册，后续升级 Plus、使用 API 都更方便。避免使用 QQ 邮箱、163 邮箱等国内邮箱，部分可能被识别拦截。
💡 注册后不知道选哪个套餐？免费版先用起来，够用就免费，不够用再升级 Plus（$20/月）解锁 GPT-5.5。
遇到 "OpenAI's services are not available in your country" 提示？
说明当前节点 IP 被 OpenAI 识别为不支持的地区。请切换到日本、新加坡或美国节点后重试。也可以清除浏览器 Cookie 后再尝试。

Section 04

ChatGPT 基础使用

注册成功后登录 chatgpt.com 即可开始使用。

界面布局：

  • 左侧边栏 — 历史对话列表，可新建、搜索、归档对话
  • 中间主区域 — 对话窗口，输入问题即可获得 AI 回答
  • 底部输入框 — 支持文字输入、文件上传、图片上传、语音输入
  • 模型切换 — 顶部可切换模型：Free/Go 默认 GPT-5.5 Instant；Plus 用户可切换 GPT-5.5 Thinking（推理模式）；Pro 用户可用 GPT-5.5 Pro

快速上手示例：

场景示例 Prompt
翻译"把以下内容翻译成英文：[你的中文内容]"
写代码"用 Python 写一个读取 CSV 文件并生成柱状图的脚本"
写邮件"帮我写一封请假邮件，原因是家中有事，请假 2 天"
学习"用简单的语言解释量子计算的基本原理"
文案"为一款智能手表写 5 条小红书风格的推广文案"
分析上传 Excel/CSV 文件 → "分析这份数据的趋势并生成图表"
高效提问技巧：提问时尽量包含具体的背景信息、期望的输出格式和长度。例如 "作为一个有 5 年经验的 Python 开发者，请 review 以下代码并给出优化建议" 比简单说 "看看这段代码" 效果好很多。

如果免费版额度不够，可以升级到 Go、Plus 或 Pro。大多数普通用户只需要在 Go（约 $8/月） 和 Plus（约 $20/月） 之间做选择——Go 提供更多 GPT-5.5 Instant 额度，Plus 解锁 GPT-5.5 Thinking 推理模式和 Codex 智能体。下面以 Plus 升级流程为例说明。

价格说明：OpenAI 按地区收费，本节参考价格为 USD 近似值（官方新加坡定价：Go S$11/月 / Plus S$30/月 / Pro S$138/月）。以 OpenAI 官网结账时显示的实时价格为准。

  1. 注册 WildCard 等虚拟信用卡平台，完成实名认证
  2. 充值对应金额（$20 + 手续费）

  4. 在支付页面填写虚拟信用卡信息

付款时注意：确保付款时的 IP 地址与信用卡注册地区一致（例如都使用美国节点），否则可能被风控拒绝。付款完成后可正常切换节点。

Section 07

高级功能与使用技巧

1. DALL-E AI 绘图

在对话中直接描述你想要的图片，ChatGPT 会通过 DALL-E 生成。例如：

  • "画一只穿着宇航服的猫站在月球表面"
  • "为我的咖啡店设计一个 Logo，极简风格，黑金配色"

2. 数据分析（Advanced Data Analysis）

上传 Excel / CSV / PDF 文件，让 ChatGPT 帮你分析数据：

  • 自动识别数据结构，生成统计摘要
  • 创建图表（折线图、柱状图、饼图等）
  • 编写并执行 Python 代码处理数据

3. GPTs 商店

浏览和使用社区创建的专用 GPT 应用，覆盖各种场景：

  • 学术写作助手 — 论文润色、文献格式化
  • 设计顾问 — UI/UX 设计建议
  • 语言教练 — 外语对话练习
  • 技术面试官 — 模拟技术面试

4. 自定义指令（Custom Instructions）

在 Settings → Personalization → Custom Instructions 中设置：

  • 告诉 ChatGPT 你的角色、背景和偏好
  • 设定默认回复风格（简洁/详细、中文/英文等）
  • 设置后每次新对话都会自动生效

5. ChatGPT API

开发者可通过 API 将 ChatGPT 集成到自己的产品中：

  • 访问 platform.openai.com 获取 API Key
  • 按 Token 用量计费：GPT-5.4 mini 当前为 $0.75/百万输入 Token，GPT-5.4 为 $2.50/百万输入 Token，GPT-5.5 为 $5.00/百万输入 Token
  • API 同样需要在支持 OpenAI 的地区网络环境下使用
你是开发者？如果你想进一步了解 API Key 获取、API 充值、按量计费，以及 Plus 和 API 的区别，可以直接看 OpenAI API 申请与计费指南。

Section 10

常见问题 FAQ

Q1：注册时提示 "OpenAI's services are not available in your country"

A：先从香港节点切换到美国、新加坡或日本节点，优先测试带“美国直连 / AI 解锁 / 原生 IP”标识的线路。切换后清除浏览器 Cookie 和缓存，使用无痕模式重试。

Q2：登录后响应特别慢或经常断开

A：通常是节点质量问题。普通问答优先日本/新加坡低延迟节点；如果是 Plus 付款、OpenAI API、Claude、Gemini 等场景，优先测试美国直连或 AI 解锁节点。使用疾风云/尔湾云的新手可用 UniClash 登录后测速切换。

Q3：ChatGPT 回答到一半突然停止

A：这是长回答的正常现象。在输入框输入"继续"或"continue"即可让它继续输出。

Q4：用了代理还是无法注册，邮箱收不到验证码

A：检查垃圾邮箱。如果使用国内邮箱（QQ/163/126），建议换用 Gmail 或 Outlook 注册。

Q5：ChatGPT Plus 支付失败

A：确保支付时的节点 IP 地区与信用卡账单地址匹配。推荐付款全程使用稳定的美国或新加坡节点，优先测试美国直连 / AI 解锁线路。反复失败时可换无痕窗口、更换卡片，或改用 iPhone 外区订阅。更完整的排查可以看 Plus / Go 充值指南。

Q6：已有订阅但看不到高级模型选项

A：确认订阅已生效（Settings → Subscription 查看状态）。如果刚订阅，稍等几分钟刷新页面。Go 套餐使用 GPT-5.5 Instant（更高额度），GPT-5.5 Thinking（推理模式）需 Plus 及以上方案。

Q7：ChatGPT App 在中国 App Store 找不到

A：ChatGPT App 仅在受支持地区的 App Store 上架。iOS 用户需使用美区 Apple ID 下载。详见我们的 手机教程中外区 Apple ID 注册部分。

Q8：ChatGPT 在中国能直接用吗？

A：不能。ChatGPT 在中国大陆和香港地区被屏蔽，必须通过科学上网工具（如 Clash）连接到日本、新加坡等受支持地区的节点后才能访问。详见本文第二节：配置网络环境。

Q9：ChatGPT 和 Claude 哪个更好用？

A：各有所长。ChatGPT 在通用对话、多模态能力、GPTs 与工具生态上更成熟；Claude 在长文本处理和代码理解方面通常表现更强。两者在国内均需科学上网，但 Claude 门槛更高：注册不支持中国手机号（需海外邮箱），且香港节点无法访问（推荐美国节点）。建议选择节点覆盖广的机场服务。

Q10：想用 GPT-5.5 最省钱的方式是什么？

A：通过 Cursor IDE 订阅（首月 5 折约 $10），可同时使用 GPT-5.5、Claude Sonnet 4.6、Codex 5.3 等多个顶级模型，综合成本往往低于单独订阅 ChatGPT Plus（$20/月）。详见第八节。

Q11：ChatGPT 免费版够日常使用吗？

A：轻度使用（偶尔提问、翻译、写简短文案）免费版够用，可以使用 GPT-5.5 Instant 有限额度。每天都用建议升级 Go（约 $8/月）或 Plus（约 $20/月，解锁 GPT-5.5 Thinking 推理模式）。详见方案对比。

Q12：ChatGPT Go 值不值得买？

A：值得，尤其适合“每天都在用，但还没必要上 Plus”的用户。Go 的价格比 Plus 低很多，但比免费版有更高的消息和工具额度。如果你主要是写文案、翻译、问问题、做轻量办公，Go 往往已经够用了。

Q13：ChatGPT Plus 和 OpenAI API 是一回事吗？

A：不是。ChatGPT Plus 是网页/App 端会员，主要提升聊天界面的模型和功能；OpenAI API 是给开发者调用模型用的，按 Token 单独计费。开通 Plus 不会自动赠送 API 额度，想开发自己的应用仍然需要单独开 API 账户并充值。更详细的说明可以看 OpenAI API 指南。
