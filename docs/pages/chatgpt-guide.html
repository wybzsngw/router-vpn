Section 01

ChatGPT 是什么

ChatGPT 是 OpenAI 开发的大型语言模型对话系统，能够理解自然语言并生成高质量文本。自 2022 年 11 月发布以来，已成为全球最受欢迎的 AI 工具之一，广泛应用于：

  • 写作辅助：撰写文章、邮件、报告、文案、翻译
  • 编程开发：代码编写、调试、代码审查、技术文档
  • 学习研究：概念解释、论文辅助、外语学习、知识问答
  • 创意设计：AI 绘图（DALL-E）、头脑风暴、内容策划
  • 数据分析：上传文件进行数据处理、图表生成、报告撰写
  • 日常助手：旅行规划、食谱推荐、职业建议等

目前 ChatGPT 的最新模型为 GPT-4o，支持文本、图片、语音多模态输入，是当前最强大的通用 AI 助手之一。

Section 03

注册 OpenAI 账号

OpenAI 在 2024 年底已大幅简化注册流程，不再强制要求海外手机号验证。以下是当前最新注册方法：

方法一：邮箱注册（推荐）

  1. 确保已连接到日本/新加坡/美国等节点
  2. 访问 chat.openai.com
  3. 点击 Sign up（注册）
  4. 输入邮箱地址（推荐使用 Gmail / Outlook / ProtonMail）
  5. 设置密码（至少 12 位，建议包含大小写字母和数字）
  6. 到邮箱中查收验证邮件，点击验证链接
  7. 填写姓名和生日信息，完成注册

方法二：第三方账号快捷登录

在注册页面可直接选择以下账号一键登录：

  • Google 账号 — 最方便，推荐
  • Microsoft 账号 — 适合已有 Outlook 邮箱的用户
  • Apple ID — 支持隐藏邮箱功能
注册建议：优先使用 Gmail 注册，后续升级 Plus、使用 API 都更方便。避免使用 QQ 邮箱、163 邮箱等国内邮箱，部分可能被识别拦截。
遇到 "OpenAI's services are not available in your country" 提示？
说明当前节点 IP 被 OpenAI 识别为不支持的地区。请切换到日本、新加坡或美国节点后重试。也可以清除浏览器 Cookie 后再尝试。
Section 04

ChatGPT 基础使用

注册成功后登录 chat.openai.com 即可开始使用。

界面布局：

  • 左侧边栏 — 历史对话列表，可新建、搜索、归档对话
  • 中间主区域 — 对话窗口，输入问题即可获得 AI 回答
  • 底部输入框 — 支持文字输入、文件上传、图片上传、语音输入
  • 模型切换 — 顶部可切换 GPT-4o / GPT-4o mini 等模型

快速上手示例：

场景示例 Prompt
翻译"把以下内容翻译成英文：[你的中文内容]"
写代码"用 Python 写一个读取 CSV 文件并生成柱状图的脚本"
写邮件"帮我写一封请假邮件，原因是家中有事，请假 2 天"
学习"用简单的语言解释量子计算的基本原理"
文案"为一款智能手表写 5 条小红书风格的推广文案"
分析上传 Excel/CSV 文件 → "分析这份数据的趋势并生成图表"
高效提问技巧：提问时尽量包含具体的背景信息、期望的输出格式和长度。例如 "作为一个有 5 年经验的 Python 开发者，请 review 以下代码并给出优化建议" 比简单说 "看看这段代码" 效果好很多。

Section 06

升级 ChatGPT Plus

ChatGPT Plus 月费 $20（约 ¥145），提供更高的使用额度和更多功能。

付款方式：

方式说明难度
国际信用卡Visa / Mastercard 国际信用卡直接支付简单（需有国际信用卡）
虚拟信用卡WildCard / Depay 等平台开卡后支付中等（需注册虚拟卡平台）
Apple Pay在 iOS ChatGPT App 中通过 Apple 订阅简单（需外区 Apple ID）
代充服务通过淘宝等平台代充最简单（但有账号风险）

使用虚拟信用卡升级步骤：

  1. 注册 WildCard 等虚拟信用卡平台，完成实名认证
  2. 充值对应金额（$20 + 手续费）
  3. 登录 ChatGPT → 左下角点击头像 → My Plan → Upgrade to Plus
  4. 在支付页面填写虚拟信用卡信息
  5. 确认支付，升级立即生效
付款时注意：确保付款时的 IP 地址与信用卡注册地区一致（例如都使用美国节点），否则可能被风控拒绝。付款完成后可正常切换节点。
Section 07

高级功能与使用技巧

1. DALL-E AI 绘图

在对话中直接描述你想要的图片，ChatGPT 会通过 DALL-E 生成。例如：

  • "画一只穿着宇航服的猫站在月球表面"
  • "为我的咖啡店设计一个 Logo，极简风格，黑金配色"

2. 数据分析（Advanced Data Analysis）

上传 Excel / CSV / PDF 文件，让 ChatGPT 帮你分析数据：

  • 自动识别数据结构，生成统计摘要
  • 创建图表（折线图、柱状图、饼图等）
  • 编写并执行 Python 代码处理数据

3. GPTs 商店

浏览和使用社区创建的专用 GPT 应用，覆盖各种场景：

  • 学术写作助手 — 论文润色、文献格式化
  • 设计顾问 — UI/UX 设计建议
  • 语言教练 — 外语对话练习
  • 技术面试官 — 模拟技术面试

4. 自定义指令（Custom Instructions）

在 Settings → Personalization → Custom Instructions 中设置：

  • 告诉 ChatGPT 你的角色、背景和偏好
  • 设定默认回复风格（简洁/详细、中文/英文等）
  • 设置后每次新对话都会自动生效

5. ChatGPT API

开发者可通过 API 将 ChatGPT 集成到自己的产品中：

  • 访问 platform.openai.com 获取 API Key
  • 按 Token 用量计费，GPT-4o 约 $2.50/百万输入 Token
  • API 同样需要在支持 OpenAI 的地区网络环境下使用

Section 09

常见问题 FAQ

Q1：注册时提示 "OpenAI's services are not available in your country"

A：切换到日本/新加坡/美国节点，清除浏览器 Cookie 和缓存后重试。使用浏览器无痕模式效果更好。

Q2：登录后响应特别慢或经常断开

A：通常是节点质量问题。尝试切换到延迟更低的节点（优先日本/新加坡），或升级到带宽更大的机场套餐。

Q3：ChatGPT 回答到一半突然停止

A：这是长回答的正常现象。在输入框输入"继续"或"continue"即可让它继续输出。

Q4：用了代理还是无法注册，邮箱收不到验证码

A：检查垃圾邮箱。如果使用国内邮箱（QQ/163/126），建议换用 Gmail 或 Outlook 注册。

Q5：ChatGPT Plus 支付失败

A：确保支付时的节点 IP 地区与信用卡账单地址匹配。推荐付款全程使用美国节点。如果反复失败，可尝试更换虚拟信用卡平台。

Q6：已有订阅但只能用 GPT-4o mini，切换不了 GPT-4o

A：确认 Plus 订阅已生效（Settings → Subscription 查看状态）。如果刚刚订阅，稍等几分钟刷新页面。

Q7：ChatGPT App 在中国 App Store 找不到

A：ChatGPT App 仅在受支持地区的 App Store 上架。iOS 用户需使用美区 Apple ID 下载。详见我们的 手机教程中外区 Apple ID 注册部分。
