Section 01

Netflix 简介与内容亮点

Netflix（奈飞）是全球最大的流媒体平台，拥有超过 2.8 亿付费用户，提供海量影视内容：

  • Netflix 原创剧集 — 鱿鱼游戏、怪奇物语、黑镜、纸钞屋、Wednesday、全裸监督等
  • 美剧经典 — 绝命毒师、纸牌屋、王冠、后翼弃兵等
  • 日剧 / 动漫 — 进击的巨人、咒术回战、间谍过家家、今际之国的闻人等
  • 韩剧 — 黑暗荣耀、僵尸校园、甜蜜家园、非常律师禹英禑等
  • 电影 — 玻璃洋葱、灰影人、亚当计划等 Netflix 独占院线级电影
  • 纪录片 — 我们的星球、社交困境、厨艺大师等

Netflix 的核心优势在于内容库因地区而异，不同国家可以看到不同的影片。通过切换节点地区，你可以解锁全球各地区的独占内容。

Section 02 · 关键步骤

前置准备：流媒体解锁节点

重要：Netflix 在中国大陆未上线，且 Netflix 对代理 IP 的检测非常严格。不是所有节点都能解锁 Netflix，你需要使用专门支持流媒体解锁的机场服务。

Netflix 对节点的要求：

  • 原生 IP / 住宅 IP — Netflix 会检测 IP 类型，普通机房 IP 大概率被封
  • DNS 解锁 — 部分机场通过 DNS 中转实现解锁
  • 专用流媒体节点 — 很多机场会标注「Netflix」「流媒体」专用节点

如何判断节点是否支持 Netflix？

  1. 连接到目标节点后，访问 netflix.com/title/80018499（测试片）
  2. 如果能正常打开并播放，说明该节点支持 Netflix
  3. 如果提示代理错误（Proxy Error）或跳转到其他地区内容，则不支持

Section 03

注册 Netflix 账号

注册流程（5 分钟完成）：

  1. 确保已连接到支持 Netflix 的节点（推荐新加坡/美国/日本）
  2. 访问 netflix.com
  3. 输入邮箱地址，点击 Get Started
  4. 设置密码
  5. 选择订阅套餐（详见下一节的价格对比）
  6. 填写支付信息（国际信用卡 / 虚拟信用卡 / Gift Card）
  7. 注册完成，开始观看

支付方式：

方式说明推荐度
国际信用卡Visa / Mastercard，绑定后自动扣款推荐
虚拟信用卡WildCard / Depay 等，充值即用推荐
Gift Card购买对应地区的 Netflix 礼品卡充值适合低价区
PayPal部分地区支持 PayPal 绑定一般
注册地区 = 账单地区：注册时所用节点的地区 + 信用卡账单地址决定了你的"Netflix 主区"。主区影响套餐价格和税费，但不影响内容库（内容库由你当前连接的节点地区决定）。

Section 05

解锁不同地区内容库

Netflix 的内容库因地区而异——切换到不同地区的节点，就能看到该地区的独占内容。这是 Netflix 最大的玩法之一。

热门地区内容特色：

美国区 内容库最大，Netflix 原创首发地，美剧最全
日本区 动漫最全，日剧独占多，吉卜力工作室全集
韩国区 韩剧首发、韩国综艺独占，韩影资源丰富
英国区 BBC 合作内容，英剧资源丰富
新加坡区 中文字幕覆盖好，亚洲内容丰富
台湾/香港区 中文字幕最全，华语内容多

切换地区的方法：

  1. 在 Clash 客户端切换到目标地区的流媒体解锁节点
  2. 刷新 Netflix 页面或重启 Netflix App
  3. 内容库会自动切换为该地区的版本
  4. 字幕和配音选项也会随地区变化

想要方便地切换多个地区？选择节点覆盖多地区的机场服务，一个订阅即可解锁全球内容：

疾风云 — 全球多地区节点 扬帆云 — 全球多地区节点 尔湾云 — 多档位流媒体套餐
Clash 分流规则技巧：在 Clash 配置中为 Netflix 域名设置专用策略组，可以做到：浏览网页走日常节点，观看 Netflix 自动走流媒体解锁节点，无需手动切换。详见 规则与分流配置。

Section 07

4K 画质与字幕设置

开启 4K Ultra HD 的条件：

条件要求
套餐Premium 套餐（支持 4K + HDR）
网速至少 25 Mbps 稳定带宽
浏览器Edge（Windows）/ Safari（macOS）/ Netflix App
显示器支持 4K 分辨率的屏幕
HDCP需支持 HDCP 2.2（外接显示器时注意）
Chrome 浏览器最高只支持 1080p！想看 4K 请使用 Edge、Netflix 桌面 App 或智能电视 App。在 Netflix 设置中将播放质量调为「高」（Settings → Playback settings → High）。

字幕设置：

  • 切换字幕 — 播放时点击对话框图标，选择可用字幕语言
  • 中文字幕覆盖 — 连接到新加坡/台湾/香港节点时中文字幕选项最多
  • 字幕样式 — 在 Netflix 账户设置 → Subtitle Appearance 中自定义字体、大小、颜色、背景
  • 双语字幕 — 安装浏览器插件 Language Reactor（原 Language Learning with Netflix），可同时显示两种语言字幕，还能逐句翻译，非常适合学习外语
学外语利器：Language Reactor 插件可以让你在看 Netflix 时同时显示中英双语字幕、单词释义、逐句暂停，把追剧变成高效的语言学习。这也是很多用户订阅 Netflix 的一大原因。

Section 10

常见问题 FAQ

Q1：Netflix 提示 "You seem to be using an unblocker or proxy"

A：当前节点 IP 被 Netflix 识别为代理。请切换到机场提供的流媒体专用节点（通常使用原生 IP 或 DNS 解锁）。如果机场没有专用流媒体节点，建议更换支持流媒体解锁的机场。

Q2：为什么连接了代理还是看不了某些剧？

A：Netflix 的内容授权按地区分配。切换到该内容上架的地区节点即可。可用 unogs.com 查询内容在哪些地区可用。

Q3：Netflix 卡顿、画质差怎么办？

A：通常是节点带宽不足。尝试切换到延迟更低、带宽更大的节点。4K 需要至少 25Mbps 稳定带宽。也可以在 Netflix 设置中将画质暂时调低。

Q4：可以同时在多少台设备上看？

A：取决于套餐。Standard 套餐支持 2 台同时在线，Premium 支持 4 台。超出数量会提示"Too many people are watching"。

Q5：Netflix 有中文配音吗？

A：部分热门 Netflix 原创内容有中文普通话配音（如鱿鱼游戏、怪奇物语），但覆盖率不高。大部分内容提供中文字幕但无中文配音。

Q6：如何在电视上看 Netflix？

A：智能电视无法直接安装代理软件。推荐在路由器层面配置透明代理（华硕路由器 / 软路由），电视连接到该路由器的 WiFi 即可直接观看。

Q7：Netflix Gift Card 在哪里买？

A：淘宝搜索"Netflix 礼品卡"可以买到各地区的充值卡，也可以在账号星球购买（支持支付宝，¥21 起）。购买时注意选择和你账号注册地区一致的礼品卡（如美区账号买美区卡）。
