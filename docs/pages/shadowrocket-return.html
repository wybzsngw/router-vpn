方法 A · 推荐

方法 A：导入机场回国专线订阅

如果你的机场提供独立的回国线路订阅链接，这是最简单的方式。

添加订阅

  1. 打开 Shadowrocket → 右上角 + → 选 Subscribe

  2. 粘贴机场提供的回国专线订阅 URL，输入备注名（如「回国线路」）→ Save

  3. 点右上角 Update 更新节点，确认回国节点已出现在节点列表

追加回国分流规则

把本站维护的回国规则集加入 Shadowrocket 的配置文件，让国内服务流量自动走回国节点：

  1. 底部 Config（配置） → 找到当前生效的 .conf 文件 → 右侧 Edit → Rules

  2. 点右上角 + → 类型选 RULE-SET，填入规则集 URL：

    https://raw.githubusercontent.com/mowei-ie/router-vpn/main/docs/return-china-rules.yaml

    策略选你的回国节点或代理组，行为选 classical

  3. 将该规则拖到规则列表最上方（优先级最高）→ Save

开启并测试

  1. 主屏幕开关打开 VPN，连接模式选 Rule（规则）

  2. Safari 访问 ip.cn → 应显示大陆 IP 归属地（因为 ip.cn 被规则集命中，走了回国节点）

  3. 打开爱奇艺 / B 站 / 央视频验证锁区内容可播放

方法 B · 手动配置

方法 B：手动添加节点 + .conf 规则

如果你有独立的回国节点链接（SS / VMess / Trojan / VLESS 格式），或者想精确控制每条规则，用 .conf 配置文件方式更灵活。

添加单条节点

  1. Shadowrocket → + → 选对应协议（SS / VMess / Trojan 等）

  2. 填入服务器地址、端口、密码/UUID、加密方式，备注填「🇨🇳 回国节点」方便识别

在 .conf 配置文件中添加回国规则

在 Shadowrocket 的 .conf 文件 [Rule] 段最前面加入（把 回国节点 替换为你实际的备注名）：

⚠️ 提示

注意事项

注意事项与兼容性

Rule-Set 格式兼容性

Shadowrocket 的 Rule-Set 语法与 Clash rule-provider 的 classical 格式不完全相同：

  • Clash 格式：每行 DOMAIN-SUFFIX,example.com（无策略字段）
  • Shadowrocket 格式：每行 DOMAIN-SUFFIX,example.com,POLICY_NAME

在 Shadowrocket 里通过 Rule-Set UI 导入时，可以在导入步骤里单独指定策略（即选到哪个代理节点），此时规则集无需携带策略名，两种格式均可兼容。如遇解析失败，退回方法 B 手动写规则。

出国 + 回国订阅共存

  • Shadowrocket 支持多个订阅同时激活，出国节点和回国节点可以并存
  • 建议把回国节点归到一个独立的代理组（Proxy Group），命名「🇨🇳 回国」，与出国代理组隔开
  • Rules 模式下，流量按规则分别走对应代理组，互不干扰

使用规则模式，不要用全局模式

主屏幕 → 代理选项 → Rule（规则） 模式才能做分流。Global（全局） 模式会把所有流量走单一节点，不适合同时管出国 + 回国。

FAQ

常见问题

Q1：Shadowrocket 在中国区 App Store 能下载吗？

不能。Shadowrocket 仅在美区 App Store 上架，售价约 $2.99。需要使用美区 Apple ID 购买，或找已购账号买断。

Q2：Malus / 快帆 App 可以和 Shadowrocket 同时开吗？

不可以。iOS 同一时间只能有一个 VPN 连接活跃，开第二个会自动断开第一个。选其中一种使用即可。

Q3：海外 VPS（CN2 GIA）能当回国节点用吗？

不能解锁锁区内容。海外 VPS 无论线路多优，出口 IP 仍在海外，爱奇艺 / 央视频等锁区内容只认真正的大陆 IP。详情见回国加速综述第六节。

Q4：配置好后可以同时翻墙出国和访问国内吗？

可以。在 Rule（规则）模式下，Shadowrocket 按规则把国内域名走回国节点，其他流量走出国代理，两条路同时工作，互不干扰。

Q5：我的机场没有回国专线，怎么获得回国节点？

两条路：① 咨询你现在使用的机场，询问是否提供双向（IEPL/IPLC 回国）线路；② 如果在大陆有家庭宽带 + 常开设备，用 Tailscale Exit Node 自建——这是唯一能解锁锁区内容的自建方案。若两条路都不具备，直接用商业加速器。
