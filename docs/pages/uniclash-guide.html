Section 01

UniClash 是什么？适合谁用？

UniClash 是部分机场服务商提供的官方/定制 Clash 客户端，登录界面会要求输入"网站代码 + 邮箱 + 密码"。它不是开源 Clash 的全平台通用版本，每家服务商都会绑定自己的网站代码（例如杨帆云填 yangfan，尔湾云填 erwan），登录后即可一键连接，不需要再手动导入订阅链接。

一句话：UniClash 适合"已经在杨帆云 / 尔湾云这类机场买了套餐、只想登录账号即用"的新手；想自己导入订阅、改规则、玩 TUN 模式，仍建议使用 Clash Verge Rev、MerlinClash、OpenClash 等通用客户端。
注意边界：不要把 UniClash 理解成"所有机场通用的官方 Clash"。不同服务商的下载页、网站代码、可用线路、客服入口都不同。本文只按已确认的杨帆云与尔湾云整理。

Section 03

UniClash 正确使用顺序：先账号，后客户端

按下面 6 步操作即可顺利登录使用，不要跳过注册和购买直接下载客户端，否则会卡在"网站代码不知道填什么""登录后没有节点"。

  1. 选支持 UniClash 的服务商：本站已确认 杨帆云、尔湾云 都有独立 UniClash 下载页。
  2. 注册账号：点上一步的服务商名称跳到注册页，按提示填邮箱和密码。
  3. 购买合适套餐：付款完成后，登录服务商官网确认账号有有效套餐和剩余流量。
  4. 下载对应 UniClash：杨帆云走 tusmartchat.com/uniclash.html，尔湾云走 tusmartchat.com/uniclash-erwan.html。
  5. 填写网站代码：杨帆云填 yangfan，尔湾云填 erwan，看到绿色 ✓ 和服务商名再继续。
  6. 输入邮箱密码登录：用刚刚注册的服务商账号，登录后点中间的电源按钮即可连接。
不要随手点"使用默认站点"："默认站点"不是杨帆云或尔湾云的账号体系，点了就会找不到你之前的购买记录，需要回来用正确的网站代码重新登录。
Section 04

网站代码怎么填？看到绿色 ✓ 才算对

打开 UniClash 后第一步就会要求填网站代码。正确填写后，输入框右侧会显示绿色对勾和服务商名称（"扬帆云"或"尔湾云"），这时再继续输入邮箱密码登录。

▲ 杨帆云：网站代码填 yangfan，绿色 ✓ 后显示"扬帆云"
▲ 尔湾云：网站代码填 erwan，绿色 ✓ 后显示"尔湾云"

填写要点

  • 只在 App 弹出"网站代码"输入框时填；如果 App 没问，直接输邮箱密码登录即可。
  • 账号密码用对应服务商官网的，不是本站账号。
  • 不确定时，不要点"使用默认站点"——那是另一个账号体系。
▲ 这个弹窗看到后请点"取消"，回去用正确的网站代码登录
误用默认站点怎么办？退出登录，重新打开 UniClash → 重新输入正确的网站代码（yangfan 或 erwan），原账号和套餐就回来了。
Section 05

Windows / Mac / Android 安装登录

三大平台都从对应服务商的下载页直接下载安装包，不要从第三方网盘或二次打包站下载，避免账号信息泄露或安装包被改。

1. Android

  1. 用浏览器打开 杨帆云 UniClash 下载页 或 尔湾云 UniClash 下载页。
  2. 选 下载 UniClash APK（线路一），速度慢就换线路二。
  3. 提示"未知来源"时，去系统设置允许当前浏览器或文件管理器安装。
  4. 下载页还提供改名 zip 版，下载后把 .zip 改回 .apk 再安装。

2. Windows（10 及以上）

  1. 建议用 Chrome / Edge 下载；如果浏览器提示"通常不会下载"，点"保留"。
  2. 双击 EXE 安装；若 Defender / 360 拦截，先关闭实时监控再装。
  3. 下载页另提供"EXE 改名 zip 版"，下载后改回 .exe 再运行。

3. Mac

  1. 先确认芯片：左上角苹果菜单 → 关于本机，看到"Apple M1/M2/M3" 就选 Mac M 芯片，看到"Intel" 就选 Mac Intel。
  2. 下载 .dmg 后拖到"应用程序"。
  3. 提示"无法验证开发者"：按住 Control 单击 App 图标 → 打开。
  4. 仍打不开：执行 sudo xattr -cr /Applications/UniClash.app 解除隔离属性。
iPhone 用户：UniClash 目前只覆盖 Android / Windows / Mac，iOS 用户请使用 Shadowrocket / Stash，需要外区 Apple ID 下载。
Section 06

登录后界面与节点切换

登录成功后界面非常简单：中间一个大圆按钮控制连接，下方显示当前线路，底部三个 Tab 分别是"首页 / 商城 / 我的"。

▲ 未连接：灰色电源按钮，点击即可启动连接
▲ 已连接：绿色按钮，下方显示当前节点，再点一次即断开

切换节点

  1. 点底部"当前线路"那一栏的箭头，弹出"选择线路"面板。
  2. 点右上角 测速，等延迟数字刷新出来。
  3. 选延迟最低（一般 60–150 ms 比较稳）的节点即可，点击后会自动切换。
  4. 解锁 Netflix / ChatGPT 等流媒体优先选择带 -NF 标记的高级节点。
▲ 选择线路：测速后挑延迟最低的节点；带 -NF 后缀的通常用于流媒体解锁
右上角"回退"按钮：点亮时表示开启自动备用切换；如果你只想固定某个节点，把它关掉即可。"规则"则用来切换全局/规则/直连模式。
Section 07

UniClash 和 Clash Verge / OpenClash 怎么选？

UniClash 偏"省心"，通用 Clash 偏"可控"，两者各有适用场景。简单的判断方法：

方案优点限制推荐人群
UniClash 登录账号即可用，无需手动导入订阅，更新和切节点都很简单。 绑定服务商生态，并非所有机场通用。 Windows / Mac / Android 新手。
Clash Verge Rev 多订阅、规则编辑、TUN 模式都灵活。 需要理解系统代理、TUN、订阅链接等概念。 桌面端长期用户、多机场用户。
MerlinClash / OpenClash 路由器全屋代理，一次配置全家用。 排障成本更高，依赖路由器性能。 软路由、华硕梅林、旁路由用户。
Shadowrocket / Stash iPhone / iPad 端最常见的 Clash 客户端。 需要外区 Apple ID 下载。 iOS 用户。
Section 08

UniClash 打不开 / 登录不上 / 全节点无信号

遇到问题不要反复重装，按下面顺序排查更省时间。

1. Windows 打不开

  • SmartScreen 安全警告：点"更多信息" → "仍要运行"。
  • 双击没反应：换 Chrome / Edge 重新下载，临时关 360、电脑管家等拦截。
  • 安装包打不开：右键属性 → 勾选"解除锁定"。
  • Win7 / Win8：UniClash 需要 Windows 10 及以上系统。

2. Mac 打不开

  • "无法验证开发者"：按住 Control 单击 App → 打开。
  • "无法检查恶意软件"：系统设置 → 隐私与安全性 → 仍要打开。
  • 磁盘映像打不开：先拖到"应用程序"，仍失败执行 sudo xattr -cr /Applications/UniClash.app。
  • M 芯片不要下 Intel 版本，反之亦然。

3. 登录后没有节点 / 全节点无信号

  • 检查是否登录的是正确的服务商账号。
  • 检查网站代码：杨帆云 yangfan，尔湾云 erwan。
  • 到服务商官网确认套餐没过期、流量没用完。
  • 更新到最新 UniClash，再测速换低延迟节点。
  • 本地网络问题：换手机热点测试，排除运营商或路由器 DNS 异常。
仍然异常：去服务商官网首页查看公告（可能正在做线路维护），或在右下角联系在线客服；路由器全屋代理用户额外检查 MerlinClash / OpenClash 插件是否有版本兼容问题。
Section 09

UniClash 常见问题

Q1：UniClash 网站代码填什么？

按服务商填。杨帆云填 yangfan，尔湾云填 erwan。如果 App 没提示网站代码，直接用服务商账号邮箱密码登录即可。

Q2：UniClash 支持 iPhone / iPad 吗？

目前两家下载页都只提供 Android / Windows / Mac。iOS 用户请改用 Shadowrocket / Stash，需要外区 Apple ID 下载。

Q3：UniClash 和 Clash Verge Rev 哪个更适合新手？

买的是支持 UniClash 的机场（杨帆云 / 尔湾云）就用 UniClash 更省心；如果你用多个机场或者要改规则、开 TUN 模式，Clash Verge Rev 更灵活。

Q4：UniClash 登录成功但没有节点怎么办？

先确认：服务商账号正确 + 网站代码正确 + 套餐未过期 + 还有剩余流量。再更新客户端 + 测速选低延迟节点；只有部分节点不可用就换其它节点或线路。

Q5：可以从第三方网盘下载 UniClash 吗？

不建议。UniClash 涉及账号登录和网络代理，请优先走服务商官方下载页，避免下载到二次打包或被植入广告 / 后门的版本。

Q6：之前点了"使用默认站点"，账号找不到了怎么办？

退出登录，重新打开 UniClash，把网站代码改回 yangfan 或 erwan，再用原邮箱密码登录即可。原服务商账号和套餐数据都还在。
