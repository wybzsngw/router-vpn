杨帆云重要通知

现在的杨帆云的订阅目前还是支持路由器上 Merlin Clash 的，目前测试有效，后续如果有任何的发现会及时的通告。

⚠ 官方通告重点（请务必阅读）

以下第三方客户端目前已不再支持，继续使用可能出现连接异常、导入失败或售后不受理等问题：

  • Clash Verge
  • Clash Meta
  • Clash For Windows
  • V2RayN / V2RayNG
  • Shadowrocket
  • Quantumult X
  • Surge / Stash / sing-box

✅ 建议使用官方客户端 UniClash

下载安装后可直接启动，登录杨帆云账号即可使用（如提示网站代码，可填写：yangfan）。

支持 Windows / Mac / 安卓。通常无需手动导入订阅，稳定性更好，排障流程更简单。
