## 写作规范（codex 必读）

每篇产品评测必须包含以下小节，**结构与 `subscription/clash-subscription-guide.md` 中各机场介绍保持一致**：

1. **产品定位**（一句话）
2. **官网与下载链接**（带 UTM 推广参数）
3. **价格表**（月付/季付/年付，含实际截图）
4. **节点覆盖与解锁能力**（哪些国内服务实测可用 / 不可用）
5. **设备数限制**（重要！海外华人家庭多设备）
6. **退款政策**（必须如实写出，影响读者决策）
7. **优惠码**（若有专属码）
8. **优缺点小结**
9. **常见问题排障**

## 合规底线（不可妥协）

- ❌ 不写"100% 解锁"、"绝对低延迟"、"永久免费"等夸大表述
- ❌ 不隐瞒已知缺点（节点拥挤、客户端 Bug、客服响应慢等）
- ❌ 不推荐无公司主体、运营时间不足 1 年的小品牌
- ✅ 在文章末尾统一注明"本站可能从注册中获得佣金，不影响你的实际价格"

## 推广链接接入流程

1. 用户（项目方）申请并取得各家联盟链接 → 提供给 codex
2. codex 将链接写入文章中"立即注册"按钮的 `href`
3. 同步在 `docs/analytics-events.js` 的 `RETURN_ACCELERATOR_DOMAINS` 白名单注册对应域名
4. 验证 GA4 后台能正确接收 `return_accelerator_click` 事件
