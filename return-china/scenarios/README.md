## 写作规范

- **每篇文章首段必须明示已知限制**（DRM、客户端检测、账号绑定等）
- **不夸大效果**：写"实测可用"或"部分内容受 DRM 限制无法解锁"
- **每篇文末配 2 个 CTA**：主推荐 1 款加速器 + 引导至横评页
- **不教用户违反平台 ToS 的操作**（如多账号共享、刷流量、虚假地理位置注册等）
