# 自建回国节点（占位）

本目录面向**有 Linux 基础的极客用户**，介绍如何利用海外 VPS + 优质回国线路（CN2 GIA / 移动 CMI / 联通 9929）部署自己的回国加速节点。

## 严肃警告（写入每篇文章首段）

⛔ **绝对不要在国内云服务器（阿里云 / 腾讯云 / 华为云等 ECS）上部署任何代理软件**

- 违反国内云厂商《用户协议》和《禁止使用条款》
- 可能触发工单告警，导致机器被封禁、账号关停
- 严重情况下涉及法律风险

✅ **正确做法**：购买**海外 VPS 中的优质回国线路产品**（落地节点本身在海外，通过 CN2 GIA / 移动 CMI 等国际优质线路与国内互联），既合规又高速。
