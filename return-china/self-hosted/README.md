# 自建回国节点

本目录介绍**自己动手搭建回国解锁方案**。在动手前，请先记住一个被很多教程带偏的技术真相。

## 一个必须先讲清的技术真相

> ⚠️ **"买海外 VPS + CN2 GIA 就能自建回国节点解锁国内视频"——这个说法是错的。**
>
> 锁区内容（爱奇艺 / 央视频 / 咪咕等）只认**真正的中国大陆 IP**。海外 VPS 无论线路多优，公网 IP 仍在海外，CN2 GIA / CMI / 9929 优化的只是链路质量，**不会**把出口 IP 变成大陆 IP。

✅ **唯一能"自建解锁锁区"的前提**：你在中国大陆有一个真实落地出口 IP。最合规、最现实的来源是**你自己（或父母 / 亲友）家里的家庭宽带**。

⛔ **不要用国内云服务器（阿里云 / 腾讯云 / 华为云 ECS）当出口**：虽然也是大陆 IP，但违反云厂商《用户协议》、会被工单封号，且带宽成本远高于商业加速器。合规的大陆 IP 来源只有家庭宽带。

## 海外 VPS 的定位（不在本目录主线）

海外 VPS（搬瓦工 / DMIT）**不能解锁锁区内容**，因此不作为"回国自建"的主方案。它们的真实价值在两处：

- **翻墙出国**：大陆访问 Google / ChatGPT / GitHub，参考本站[翻墙出国订阅指南](https://www.aixiaobai168.com/clash-subscription-guide)。
- **低延迟中转（非锁区）**：海外访问不做地理封锁的大陆资源（SSH、企业系统、普通网站）的链路加速。

选购建议见[回国加速完整指南](https://www.aixiaobai168.com/pages/return-china-guide)第七节。

## 在大陆没有落地点？

那自建这条路不成立，请直接用商业回国加速器（本质是"按月租用别人维护好的大陆 IP 池"），参考[回国加速完整指南](https://www.aixiaobai168.com/pages/return-china-guide)。

## 写作规范

- **必须**诚实区分"解锁锁区（需大陆 IP）"与"链路加速（海外 VPS 可做）"，不得混淆误导
- **必须**给出可复现的配置步骤与验证方法（如查 IP 归属地）
- **必须**标注合规边界，不写成软文
- 涉及联盟链接时，语境必须与产品真实能力一致（VPS 不挂"解锁锁区"语境）
