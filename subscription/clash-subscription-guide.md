## 一、什么是机场/订阅

### 1.1 机场

**机场**是中文圈对代理服务商的俗称，因其提供「起飞」（翻墙）能力而得名。机场通常提供：

- **订阅链接（Subscription URL）**：一个 URL，Clash 等客户端通过它自动获取节点列表和配置
- **多协议支持**：Trojan、VMess、VLESS、Shadowsocks 等
- **多地区节点**：香港、日本、美国、新加坡等

### 1.2 订阅

**订阅**指通过订阅链接自动获取并更新代理节点配置的过程。订阅链接通常返回：

- **Base64 编码的节点列表**：Clash 原生订阅格式
- **或 YAML 格式的 Clash 配置**：包含节点、规则、策略组等

### 1.3 常见协议

| 协议 | 特点 | 适用场景 |
| ---- | ---- | -------- |
| **Trojan** | 伪装成 HTTPS，抗封锁强 | 推荐首选 |
| **VMess** | 灵活，可配置性强 | 通用 |
| **VLESS** | 轻量，VMess 简化版 | 低延迟 |
| **Shadowsocks** | 成熟稳定 | 老设备 |
| **Hysteria** | 基于 QUIC，高速度 | 大带宽需求 |

## 二、如何选择机场

选择机场时，可从以下维度考虑：

| 维度 | 说明 |
| ---- | ---- |
| **稳定性** | 是否经常宕机、节点是否频繁更换 |
| **速度** | 带宽、延迟、丢包率 |
| **节点覆盖** | 地区数量、是否支持流媒体解锁 |
| **价格** | 月付/季付/年付价格，流量包大小 |
| **流媒体解锁** | Netflix、Disney+、ChatGPT 等是否可用 |
| **客服支持** | 工单响应、Telegram 群组 |

## 五、通用申请要点

- 优先选择支持 **Clash 订阅** 与 **HTTPS** 的机场
- 留意是否提供「订阅转换/短链」与「流媒体解锁/ChatGPT」
- 保存售后渠道（客服/工单/Telegram 群）
- 订阅链接一般可在「我的订阅 / 账户 / 仪表盘」中找到
- 导入后如节点为空，先在机场后台「重置订阅」再刷新

## 六、订阅获取步骤

1. **注册账号**：访问上述官网，完成注册并完成邮箱验证（如需要）。
2. **购买套餐**：选择适合的月付/季付/年付套餐并完成支付。
3. **进入控制台**：登录后进入「控制台」「订阅」「我的订阅」等页面。
4. **复制订阅链接**：找到「Clash 订阅」「订阅链接」等选项，复制完整 URL。
5. **导入 Clash**：在 Clash 客户端中粘贴订阅地址，或使用订阅转换服务生成 Clash 配置。

## 七、将订阅导入 MerlinClash（华硕梅林）

1. 进入路由器管理界面 → 软件中心 → MerlinClash → 打开「订阅设置」。
2. 在「订阅链接」粘贴机场提供的 Clash 订阅 URL（`https://...`）。
3. 点击「更新订阅」→ 等待节点同步完成。
4. 到「运行状态/节点列表」检查是否加载成功；如失败，重试「更新订阅」。

> 若需分流/解锁流媒体，可在「规则设置」启用对应的规则集或策略组。

## 八、注意事项

1. **不要分享订阅链接**：订阅链接包含你的账号信息，泄露可能导致账号被他人使用。
2. **定期更新**：节点会变动，建议定期刷新订阅或使用订阅自动更新。
3. **备份配置**：重要节点可保存为本地 YAML 文件，以防订阅失效。
4. **合规使用**：遵守当地法律法规，仅用于合法用途。
